# Network Traffic Analysis using Wireshark

## Objective
To capture and analyze network packets to identify suspicious activity.

## Tools Used
- Wireshark

## Steps Performed
1. Captured live network traffic
2. Applied filters (HTTP, DNS, TCP)
3. Identified unusual packets

## Sample Filters (Commands)
tcp
http
dns

## Findings
- Detected DNS requests
- Observed TCP handshake
- Identified suspicious IP traffic
